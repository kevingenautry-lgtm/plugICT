FAQ

Questions, answered.

Do I need to be technical to set this up? ＋

No. You run one setup file that does everything, then paste a small config into your AI agent. If anything's off, a built-in --doctor command tells you exactly what to fix in plain English.

Does it work without internet? ＋

The vault itself runs entirely on your machine — searching never calls a cloud API. Your first search downloads two small AI models once; after that the vault is fully offline. (Your AI agent, like Claude Desktop, uses its own internet connection as normal.)

Can I use it with ChatGPT or Claude on the web? ＋

Today it connects to local AI apps — Claude Desktop, Cursor, Hermes and other MCP-compatible agents. A hosted connector for web AIs like ChatGPT and Claude.ai is on the roadmap, and lifetime buyers get first access.

What exactly do I download? ＋

An encrypted knowledge vault plus a lightweight app. Your personal license.key (emailed to you) unlocks it. The license is tied to you and traceable, so please don't share it.

Windows, macOS or Linux with Python 3.9+. The setup script handles the rest in an isolated environment, so it won't interfere with anything else on your computer.
