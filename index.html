Questions, answered.

Do I need to be technical to set this up? +

No. You download, drop in your license key, and double-click the setup file — it handles everything. Then you paste one small config into your AI agent (copy-paste, no code). If anything's off, a built-in doctor check tells you exactly what to fix, in plain English.

Does it work without internet? +

The vault's search runs locally on your machine — it never sends your queries to a cloud. But your AI agent and the YouTube links do need internet. Think of it like this: the library is on your shelf, but you need wifi to ask the librarian.

Can I use it with ChatGPT or Claude on the web? +

It works today with Claude Desktop (free download), Cursor (free) and Hermes Agent (free) — any local MCP agent. Web-based ChatGPT and Claude don't support MCP yet; a hosted connector for web AIs is on the roadmap.

What exactly do I download? +

Three things: an encrypted knowledge vault file, a setup file that handles everything automatically, and your personal license.key (emailed to you). No app, no installer, no bloat. The license is tied to you and traceable, so please don't share it.

Do I need to pay for anything else? +

The vault is one-time $23.99 with zero ongoing fees. Your AI agent needs its own API key — free options include Google Gemini (1,500 queries/day free), and DeepSeek runs about $2/month. Claude Desktop's free plan works too.

How can I pay? +

DuitNow QR and FPX online banking for Malaysia, international cards via Stripe, or USDT (Solana / TRC20) for crypto. However you pay, your license is emailed to you and the product works exactly the same.

What are the system requirements? +

Windows, macOS or Linux with Python 3.9+. The setup script handles the rest in an isolated environment, so it won't interfere with anything else on your computer.
